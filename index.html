Google Maps API キーを入力して始める

🔒 プライバシー保護：入力されたキーはブラウザの localStorage にのみ保存され、外部サーバーへ送信されることは一切ありません。
  1. Google Cloud Console を開き、プロジェクトを作成（または選択）する
  2. 左メニュー「APIとサービス」→「ライブラリ」を開く
  3. Maps JavaScript API を検索して「有効にする」
  4. 同様に Street View Static API も有効にする
  5. 「APIとサービス」→「認証情報」→「認証情報を作成」→「APIキー」
  6. 発行されたキーをコピーして上の欄に貼り付ける
💳 請求先アカウント（クレジットカード）の登録が必須ですが、月 $200 分の無料枠があるため個人利用では通常無料の範囲内に収まります。

🔒 キーの保護：Cloud Console でキーをクリック →「アプリケーションの制限」→「ウェブサイト」に file://* や使用するドメインを追加すると他者に悪用されにくくなります。

💾 入力したキーはこのブラウザの localStorage に保存されます。ファイルを再度開いても入力不要です。
